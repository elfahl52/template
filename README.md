## 📋 Description
A beautiful, modern landing page featuring a stunning mountain hero section, interactive cards, and a professional footer. Perfect for personal portfolios, business websites, or creative agencies.

---

## ✨ Features
- ✅ **Responsive Design** - Works perfectly on all devices
- ✅ **Hero Section** - Eye-catching mountain background with CTA buttons
- ✅ **Navigation Menu** - Clean header with smooth navigation
- ✅ **Image Gallery** - Three-column card layout with hover effects
- ✅ **Contact Section** - Professional footer with contact information
- ✅ **Smooth Animations** - Modern transitions and interactions
- ✅ **Semantic HTML5** - Clean, accessible code structure
